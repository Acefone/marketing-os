## How to use this template

Copy the brief below and fill in every required field before submitting to the Content Writer Agent. Optional fields add quality — include them when you can.

The more specific the brief, the better the output. A vague brief produces a generic draft.

The Content Writer Agent will reject the brief if any of these are missing:

- [ ] Target persona specified  
- [ ] Funnel stage selected  
- [ ] Reader's problem described  
- [ ] What the reader should do after reading
- [ ] Primary keyword (SEO blogs only)  
- [ ] Key differentiator identified  
- [ ] Unique angle stated  
- [ ] CTA selected
