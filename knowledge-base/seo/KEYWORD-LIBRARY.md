## How to use this file

Each cluster below maps a topic to a set of target keywords with intent, difficulty, and content recommendations. The SEO Agent reads this file to:
1. Avoid recommending keywords already well-covered by existing Acefone content
2. Build on existing cluster data rather than starting from scratch each week
3. Flag cannibalisation risks when multiple briefs target the same cluster

**To add a new cluster:** Copy the template at the bottom of this file and paste it at the appropriate section.
